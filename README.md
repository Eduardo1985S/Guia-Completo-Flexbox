```css
  .container { flex-direction: column; }
}
```

## 🤔 Dúvidas Comuns

### **"Quando usar Flexbox vs Grid?"**
- **Flexbox** → Para layouts em **1 dimensão** (linha OU coluna)
- **Grid** → Para layouts em **2 dimensões** (linhas E colunas)

### **"Por que meus itens não ficam centralizados?"**
- **Horizontal** → Use `justify-content: center`
- **Vertical** → Use `align-items: center`
- **Ambos** → Use os dois juntos!

### **"Como fazer itens ocuparem espaço igual?"**
- Use `flex: 1` em todos os itens
- Teste no guia interativo para ver!

   - Alinhamento de múltiplas linhas

### 🚀 **Nível Intermediário-Avançado - Propriedades dos Itens**

8. **Flex Grow** 📈
   - Como itens crescem no espaço disponível
   - **Teste Interativo**: Sliders 0-5 para cada item
   - **Dica Educativa**: Distribuição proporcional de espaço

9. **Flex Shrink** 📉
   - Como itens encolhem quando necessário
   - **Teste Interativo**: Container estreito para forçar shrink
   - **Dica Educativa**: Controle de encolhimento inteligente
